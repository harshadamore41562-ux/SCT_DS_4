## Task 04: Traffic Accident Analysis  

### 📌 Overview  
This project is part of my Skill Craft Data Science internship.  
It focuses on analyzing **Indian traffic accident data** to identify patterns related to road conditions, weather, and time of day.  
The goal is to visualize accident hotspots and contributing factors to gain insights into road safety challenges in India.  

---

### 🛠️ Tools Used  
- Python  
- pandas  
- matplotlib  
- seaborn  
- scikit-learn (for exploratory modeling, if required)  

---

### 📁 Files in Repository  
- `accident_dataset.csv` → Dataset containing traffic accident records in India  
- `Task04_Accident_Analysis.py` → Python script for preprocessing, analysis, and visualization  
- `README.md` → Project documentation  
- `accident_hotspots.png` → Visualization of accident hotspots  
- `accident_factors.png` → Visualization of contributing factors (road, weather, time)  

---

### 📋 Steps Performed  
1. Loaded dataset (`accident_dataset.csv`) using pandas.  
2. Cleaned and preprocessed data (handling missing values, encoding categorical variables).  
3. Explored accident patterns by road conditions, weather, and time of day.  
4. Visualized accident hotspots using heatmaps and bar charts.  
5. Identified contributing factors and trends in accident severity.  
6. Summarized findings for actionable insights.  

---

### 📊 Output & Results  
- **Accident Hotspots:** Visualized using heatmaps and charts  
- **Contributing Factors:** Road conditions, weather, and time of day  
- **Insights:** Patterns that highlight risk areas and conditions in Indian traffic accidents
